创建新的功能规格文档。

步骤:
1. 读取 `docs/` 目录，找到当前最大的 SPEC 序号
2. 新序号 = 最大序号 + 1
3. 向用户确认功能名称和简述 (如果用户未在命令参数中提供)
4. 在 `docs/` 下创建 `SPEC-{序号}-{功能名}.md`，使用以下模板:

```markdown
# SPEC-{序号}: {功能标题}

> 起草日期: {今天日期}
> 状态: 草稿

## 背景

{为什么需要这个功能}

## 目标

{要达到什么效果}

## 方案

### 前端改动

### 后端改动

### 数据结构变更

## 验收标准

- [ ] {标准1}
- [ ] {标准2}
```

5. 报告创建结果，输出文件路径
