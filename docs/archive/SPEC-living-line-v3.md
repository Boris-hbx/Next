# Living Line V3 - 呼吸位置渐进规范

## 概述

呼吸效果从左侧开始，每次呼吸后位置向右移动一步，5次完成从左到右的完整过程，然后循环。

### 参数配置

| 参数 | 值 | 说明 |
|------|-----|------|
| 呼吸步数 | 5 | 从左到右分5步 |
| 起始位置 | 10% | 屏幕左侧 10% |
| 结束位置 | 90% | 屏幕右侧 90% |
| 单次周期 | 3.5秒 | 每次呼吸（隆起+回落） |
| 完整周期 | 17.5秒 | 5次呼吸走完全程 |

### 位置计算

```javascript
// 5个呼吸位置
const breathPositions = [0.10, 0.30, 0.50, 0.70, 0.90];

// 当前步骤（0~4）
let breathStep = 0;

// 每完成一次呼吸，步骤+1
if (breathPhase > Math.PI * 2) {
    breathPhase -= Math.PI * 2;
    breathStep = (breathStep + 1) % 5;  // 循环 0→1→2→3→4→0
}

// 当前呼吸中心位置
const breathCenterX = width * breathPositions[breathStep];
```

### 时间线

```
|-------|-------|-------|-------|-------|
  左10%   30%     50%     70%    右90%   回到左10%
```

## 其他效果保持不变

- **光晕**：2倍膨胀效果
- **涟漪**：鼠标靠近时缓慢扩散
- **磁吸**：鼠标靠近底部时线条被吸起
