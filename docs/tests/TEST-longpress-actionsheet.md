|----|------|---------|---------|
| LP-01 | 待办条目长按触发 | 扁平列表中长按一个待办条目 500ms | 震动反馈 + 底部弹出操作菜单 |
| LP-02 | 例行条目长按触发 | 长按一个例行条目 500ms | 震动反馈 + 底部弹出操作菜单 |

| LP-77 | iOS Safari | iOS Safari 长按 | 不触发系统文本选择/放大镜 |

## 七、已知风险点

| ID | 风险 | 影响 | 缓解措施 |
|----|------|------|---------|
| R-01 | drag.js 300ms 与 ActionSheet 500ms 竞争 | 移动端待办可能同时触发拖拽和菜单 | 扁平列表中不挂拖拽 handler |
| R-02 | iOS Safari 长按触发系统菜单 | 系统菜单和自定义菜单同时出现 | CSS `-webkit-touch-callout: none` + `user-select: none` |
| R-03 | 动态渲染后事件丢失 | 列表重新渲染后新条目没绑定长按 | 使用事件委托或重新渲染后 rebind |
