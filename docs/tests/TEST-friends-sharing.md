# 测试用例: 好友与分享
> 更新日期: 2026-02-24

## 已知 Bug

### BUG-2: 分享横幅中文被 Unicode 转义 [P2]

**现象**: `friends.js` 中 `renderShareBanners()` 使用了 `\uXXXX` 转义字符，实际显示可能正常但代码可读性差

**修复**: 将 Unicode 转义改回中文原文

### BUG-5: 分享弹窗无 loading 状态 [P3]

**现象**: `renderShareFriendsList()` 懒加载好友列表时，可能先闪现"还没有好友"再更新

**修复**: 加载中显示 spinner

## 测试用例

| # | 用例 | 前置条件 | 操作步骤 | 预期结果 |

### 4. 边界与异常

| # | 用例 | 前置条件 | 操作步骤 | 预期结果 |
