| XM-07 | 空数据库 | 新用户问"我有什么任务" | 摘要全部为 0，Claude 回答"还没有数据" |

## 八、已知风险

| ID | 风险 | 影响 | 缓解措施 |
|----|------|------|---------|
| R-01 | 学习笔记内容太长 | update_english_scenario 传入长内容可能超 Claude max_tokens | 工具 description 标注内容上限 |
| R-02 | 记账货币识别错误 | "花了 15 块"可能被识别为 CAD | system prompt 加默认货币规则 |
| R-03 | 多条匹配时 Claude 自行选择 | 用户说"删掉那个"，匹配到多条 | prompt 指导：多条匹配时列出让用户选 |
| R-04 | query 返回太多数据 | 用户有 500 条记账，query_expenses 全返回 | 默认 limit 20，工具定义中标注 |
