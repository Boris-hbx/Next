# TEST-trip 差旅模块测试用例

> 覆盖范围: 差旅(Trip)模块 — 列表、详情、条目CRUD、照片、协作、弹窗、手机端适配

---

## 1. 生活 Hub 入口

| # | 测试步骤 | 预期结果 |
|---|---------|---------|
| 1.1 | 进入"生活"页面 | Hub 中显示"差旅 ✈️"卡片 |
| 1.2 | 点击"差旅"卡片 | 进入差旅列表页，显示 header("← 生活 差旅") |
| 1.3 | 点击"← 生活" | 返回生活 Hub |
| 1.4 | 再次进入差旅，刷新页面 | 自动恢复到差旅列表（localStorage 持久化） |

## 2. 列表页

| # | 测试步骤 | 预期结果 |
|---|---------|---------|
| 2.1 | 无行程时查看列表 | 显示空状态"还没有差旅行程"+ 提示 |
| 2.2 | **[手机]** 查看列表页底部 | FAB "+"按钮可见，不被底部导航栏遮挡 |
| 2.3 | 有行程时查看列表 | 按"进行中/即将出发"和"已结束"分组显示 |
| 2.4 | 查看行程卡片 | 显示标题、目的地、日期范围、条目数、金额、报销进度条 |
| 2.5 | 共享行程卡片 | 显示"共享"标签 |
| 2.6 | 点击行程卡片 | 进入行程详情页 |

## 3. 新建行程弹窗

| # | 测试步骤 | 预期结果 |
|---|---------|---------|
| 3.1 | 点击 FAB "+" | 弹出"新建行程"弹窗 |
| 3.2 | **[手机]** 弹窗显示 | 弹窗应近全屏显示，完全遮盖底层内容（header 不透出来） |
| 3.3 | **[手机]** 弹窗内容 | 可滚动，底部按钮（取消/保存）可点击，不被底部导航遮挡 |
| 3.4 | **[手机]** 点击弹窗外半透明区域 | 关闭弹窗 |
| 3.5 | 不填标题点保存 | 提示"请输入标题" |
| 3.6 | 填完标题+日期点保存 | 创建成功，自动跳转行程详情 |
| 3.7 | 点"取消" | 关闭弹窗，不创建 |
| 3.8 | **[桌面]** 弹窗居中显示 | 弹窗宽度约 480px，居中于页面，背景半透明遮罩 |

## 4. 行程详情页

## 5. 添加/编辑条目

| # | 测试步骤 | 预期结果 |
|---|---------|---------|

| 5.8 | Owner 删除条目 | 确认后删除成功 |
