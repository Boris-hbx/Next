# SPEC: 侧边栏折叠时完全隐藏内容

## 问题

已完成/已删除侧边栏在折叠状态（300px）时，内容部分露出来了，显示被截断的任务名和操作按钮，视觉效果差。

## 实现方案

### CSS 修改

折叠状态下，内容区域使用 `display: none` 或 `visibility: hidden` + `height: 0`：

```css
/* 侧边栏区块 */
.sidebar-section {
    width: 300px;
    transition: width 0.3s ease;
    overflow: hidden;
}

.sidebar-section.expanded {
    width: 666px;
}

/* 内容区域：折叠时隐藏 */
.sidebar-section .section-content {
    display: none;
}

.sidebar-section.expanded .section-content {
    display: block;
}
```

### 关键点

1. **折叠时完全隐藏**: section-content 在非 expanded 状态下 `display: none`
2. **展开时显示**: 只有 `.expanded` 状态才显示 section-content
3. **header 始终可见**: section-header 在任何状态都可见可点击

## 验收标准

1. 折叠状态下，只显示标题栏（✓ 已完成 ▶）
2. 折叠状态下，任务列表完全不可见
3. 点击标题栏后，平滑展开显示内容
4. 收起时内容平滑隐藏
