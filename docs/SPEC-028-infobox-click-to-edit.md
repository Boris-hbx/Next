# SPEC-028: Infobox 点击内容直接编辑

> 起草日期: 2026-01-07
> 状态: 已完成

## 背景

当前任务详情 Infobox 中，用户需要先点击"编辑"按钮才能进入编辑模式修改任务内容（content）。这增加了不必要的操作步骤，降低了用户体验。

## 目标

点击任务的 content 区域时，自动进入编辑状态，无需额外点击"编辑"按钮。

## 当前行为

1. 用户点击任务项，打开 Infobox
2. 用户看到任务内容（只读状态）
3. 用户点击"编辑"按钮
4. 进入编辑模式，可以修改内容
5. 点击"保存"保存修改

## 期望行为

1. 用户点击任务项，打开 Infobox
2. 用户点击 content 区域
3. **直接进入编辑模式**，光标定位到输入框
4. 用户修改内容
5. 点击"保存"或按回车保存

## 技术方案

在 content 显示区域添加 `onclick` 事件，触发编辑模式：

```javascript
// 点击 content 区域进入编辑模式
function onContentClick(taskId) {
    enterEditMode(taskId, 'content');
}
```

## UI 变更

1. Content 区域鼠标悬停时显示视觉提示：
   - 背景变为淡色（如 rgba(102, 126, 234, 0.05)）
   - 光标变为文本输入样式（cursor: text）

2. 可选：悬停时在右上角显示小铅笔图标
