# SPEC-026: 例行面板确认按钮

> 起草日期: 2026-01-07
> 状态: 已完成

## 背景

例行面板（点击"例行"按钮弹出）需要添加一个"确认"按钮。

## 确认的设计

### 1. 按钮位置
**底部，紧邻 + 按钮**

### 2. 按钮功能
**保存并关闭面板**

### 3. 按钮样式
- 颜色: 蓝色 (#667eea)
- 效果: 毛玻璃 (backdrop-filter: blur)
- 大小: 比普通按钮大一点、长一点

## 示意图

```
┌─────────────────────────────────────┐
│ 🔥 你真是强的可怕              [×]  │
├─────────────────────────────────────┤
│  □ 例行任务1                        │
│  □ 例行任务2                        │
├─────────────────────────────────────┤
│ [添加例行任务...]    [+] [  确认  ] │
└─────────────────────────────────────┘
```
